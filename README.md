# Myles.GitHub.io

This is the source code for [my GitHub Page](http://myles.github.io/) and for [my archive organizations GitHub Page](http://myles.github.io/archive/).

## Usage

**Build** the website:

```shell
$ yarn build
```

**Start** the development server:

```shell
$ yarn start
```

**Deploy** the web site to GitHub Pages:

```shell
$ yarn deploy
```

## Develop

```shell
$ yarn install
```
